NpLogin is custom WP Login plugin. You can view the customized Wordpress login screen once you install this plugin.

== Upgrade Notice ==

This is the first version of NpLogin plugin

== Description ==

To change the look of WordPress login page you can use this plugin.

== Installation ==

You can either install it automatically from the WordPress admin, or do it manually:

1. Unzip the archive and put the 'NpLogin' folder into your plugins folder (/wp-content/plugins/).
1. Activate the plugin from the Plugins menu.

= Usage =

Once installed you can see the customized Wordpress login page 
Go to *WP-Admin -> Settings -> NP Login* for configuration.

= Changing the CSS =

If you need to configure the CSS style of NpLogin, you can copy the 'NpLogin-4-themes.css' file from the plugin directory to your theme's directory and make your modifications there. This way, you won't lose your changes when you update the plugin.

== Frequently Asked Questions ==

= How can I contribute for NpLogin?  =

If you want to contribute for NpLogin then feel free to contact us through http://www.dosurfin.com/contact-us/

= How to use this plugin on my theme? =

You will find two files along with this plugin. Copy NpLogin-4-themes.php, NpLogin-4-themes.css and images to your theme's directory. If your theme's name is mytwentyeleven then you can copy those files to example.com/wp-content/themes/mytwentyeleven/ and include 'NpLogin-4-themes.php' inside your theme's function file.
